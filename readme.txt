Система контроля задач

база устанвливается отдельным скриптом

набор зависит от уровня/ранга пользователя

возможности админа
-создать/удалить пользователя
-создать/удалить группу
-все, что может руководитель

возможности руководителя
-создать/удалить/открыть задачи в своих группах
-назначить задачу исполнителю

возможности исполнителя
-закрыть задачу

группа состоит из:
-дата создания
-название
-описание
-список пользователей входящих в нее

задача состоит из:
-заголовок
-описание
-создатель
-время создания
-время выполнения
-прилагаемые картинки
-комментарий
-важность
-состояние(не назначена,открыта, возвращена, завершена, переоткрыта)

пользователь состоит из:
-дата создания
-логин
-пароль
-ранг
-список задач, в которых он учавствует

все панели и окна которые может видеть пользователь:
-задачи, назначенных ему
-задачи назначенных другому пользователю
-все задачи, для каждой из групп в которых он состоит
-пользователи входящих в любую группу, в которую входит текущий пользователь
-группы в которые входит пользователь


порядок отображения панелей:
-панель задач текущего пользователя
-панель пользователей
-панель групп
-панель задач выбранного пользователя

панель задач состоит из:
-номер задачи
-название
-автор задачи
-приоритет
-состояние
-дата
